Apa itu FLOWS?

FLOWS merupakan sebuah sistem deteksi dini banjir berbasis Internet of Things (IoT) yang dirancang untuk meningkatkan kesiapsiagaan terhadap potensi banjir secara otomatis dan real-time. Sistem ini mengintegrasikan sensor jarak air, sensor intensitas hujan, dan sensor kelembaban tanah yang diolah menggunakan fuzzy logic untuk menentukan status kondisi lingkungan (AMAN, WASPADA, dan BAHAYA), serta memberikan informasi peringatan dini guna meminimalkan risiko dan dampak banjir.

FEATURE FLOWS

Pemantauan Lingkungan Secara Real-Time

FLOWS memantau kondisi lingkungan secara langsung melalui sensor ketinggian air, intensitas hujan, dan kelembaban tanah untuk mendeteksi perubahan yang berpotensi menyebabkan banjir.

Analisis Keputusan Berbasis Fuzzy Logic Sugeno

Data dari seluruh sensor diolah menggunakan metode fuzzy logic untuk menentukan status kondisi secara adaptif, yaitu AMAN, WASPADA, atau BAHAYA, sehingga keputusan yang dihasilkan lebih akurat dan tidak kaku.

Sistem Peringatan Dini Otomatis

FLOWS memberikan peringatan dini secara otomatis ketika terdeteksi kondisi WASPADA atau BAHAYA melalui indikator visual dan audio guna meningkatkan kesiapsiagaan dan mengurangi risiko banjir.
